# Команды запуска для Railway

## Стандартная команда (рекомендуется)
Railway автоматически использует команду из `package.json`:
```bash
npm start
```

Это эквивалентно:
```bash
node dist/index.js
```

## Альтернатива с PM2 (для многопроцессной обработки)

Если нужна многопроцессная обработка (аналог `gunicorn -w 4`), можно использовать PM2:

1. Добавьте PM2 в зависимости:
```bash
npm install pm2 --save
```

2. В Railway установите Start Command:
```bash
npm run start:pm2
```

Или напрямую:
```bash
pm2 start dist/index.js -i 4 --name mariko-backend
```

Где `-i 4` означает запуск 4 процессов (workers).

## Важно

- **Для большинства случаев достаточно стандартной команды `npm start`**
- PM2 нужен только если требуется обработка большого количества запросов
- Railway автоматически определит команду из `package.json`, если не указать Start Command вручную
